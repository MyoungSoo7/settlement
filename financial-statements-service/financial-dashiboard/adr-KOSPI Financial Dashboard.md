프로젝트명 : KOSPI Financial Dashboard

목적
- 코스피 상장기업의 최근 5개년 재무제표를 조회한다.
- 기업 검색 및 재무정보를 시각화한다.
- OpenDART 기반 데이터를 주기적으로 수집하여 최신 상태를 유지한다.

대상 사용자
- 개발자
- 투자자
- 취업 준비생
- 기업 분석 사용자

기능 목록
F-001 기업 재무제표 조회
- 코스피 기업의 최근 5개년 재무제표 조회
- 매출, 영업이익, 순이익, 자산, 부채, 자본 표시

F-002 경제동향 스크랩
- 주요 경제 뉴스 자동 수집
- 키워드: 금리, 환율, 물가, 반도체, 배터리, 조선, 자동차, AI, 부동산
- 출처: 한국은행, 통계청, 금융위원회, 네이버뉴스 RSS, 공공데이터

F-003 기업별 관련 뉴스 매핑
- 기업명, 종목코드, 업종 키워드 기준으로 뉴스 연결
- 예: 삼성전자 → 반도체, 메모리, AI, 환율
- 예: 현대차 → 자동차, 전기차, 환율, 미국 판매

F-004 AI 경제동향 요약
- 수집된 뉴스 본문을 요약
- 긍정 요인 / 부정 요인 / 중립 이슈 분류
- 기업 재무제표와 연결하여 해석

F-005 대시보드
- 기업 재무제표
- 최근 뉴스
- 경제동향 요약
- 산업별 키워드 트렌드
- 투자 참고용 리스크 요약

- F-001 기업 검색
  기업명 또는 종목코드로 기업을 검색한다.
  입력 : 기업명, 종목코드
  출력 :기업명,  종목코드, 시장구분,  업종
  완료조건 : 검색어 입력 시 자동완성 제공, 기업 상세 페이지 이동 가능
- F-002 최근 5년 재무제표 조회
  기업의 최근 5개년 재무제표를 조회한다.
- 표시 항목 :매출액, 영업이익,   당기순이익,  총자산,  총부채,  자본총계,  영업이익률,  ROE,  ROA
  완료조건 :5개년 데이터가 표 형태로 출력된다.
- F-003 재무 차트
  재무 데이터를 시각화하여 표시한다.
  완료조건 : 차트 형태로 재무 데이터가 출력된다.
- F-004 기업 비교
  최대 3개 기업을 선택하여 재무지표를 비교한다.
  비교항목 : 매출, 영업이익, 순이익, ROE, ROA 
- F-005 데이터 수집 배치
  OpenDART 기반 데이터를 주기적으로 수집하여 최신 상태를 유지한다.
  주기 : 매일 새벽 2시
  처리 절차 : 기업목록 조회 -> 사업보고서 조회 -> 재무제표 조회 -> DB저장 -> 로그 저장
  완료조건 : 데이터 수집 배치가 정상적으로 수행된다.
