# 거시 신호 해석 (economics-service 4대 지표)

경제지표는 **종목이 아니라 "환경"을 판단**하는 데 쓴다.
좋은 기업도 비우호적 환경에서는 진입 시점을 늦추는 것이 초보자 기준이다.

## 지표 코드 (econ_latest / econ_series 의 code)

| code | 이름 | 단위 | 주기 |
|---|---|---|---|
| `BASE_RATE` | 한국은행 기준금리 | % | 일별 |
| `TREASURY_3Y` | 국고채 3년 금리 | % | 일별 |
| `USD_KRW` | 원/달러 환율 | KRW | 일별 |
| `CPI` | 소비자물가지수 | 2020=100 (지수) | 월별 |

`econ_latest()` 는 4개 전부의 최신값(latest.observedDate/value)과
전기 대비 변동(change.amount/ratePercent)을 한 번에 준다.

## 지표별 해석 규칙

### 기준금리 (한국은행)
- **인하 사이클** (최근 2회 이상 인하 또는 인하 후 동결): 주식 우호 — 할인율 하락, 유동성 확대
- **동결 지속** (6개월 이상): 중립 — 방향 전환 신호를 기다리는 구간
- **인상 사이클**: 비우호 — 밸류에이션 압박, 특히 성장주·고부채 기업에 부담
- 최근 3~6개월 시계열로 사이클을 판정하라. 마지막 값 하나로 판단 금지.

### 국고채 3년 금리
- 기준금리보다 **선행**한다: 국고채3년 < 기준금리 = 시장이 인하를 기대 (완화 기대)
- 국고채3년이 기준금리 대비 크게 위 = 시장이 추가 인상/긴축 장기화를 기대
- 스프레드(국고채3년 − 기준금리)의 부호와 최근 방향을 함께 보고하라.

### USD/KRW 환율
- **급등 구간** (3개월 내 +5% 이상): 외국인 자금 이탈 압력 — 지수 전반에 부담
- **안정/완만한 하락**: 외국인 수급 우호
- **이중성 명시 필수**: 원화 약세는 수출 기업(전자·자동차 등) 실적에는 우호적.
  종목의 수출 비중에 따라 해석이 반대가 됨을 항상 말하라.

### CPI (소비자물가지수)
- 상승률 둔화(디스인플레이션) = 금리 인하 여지 = 우호
- 재가속 = 긴축 장기화 위험 = 비우호
- CPI 는 지수 원값이 아니라 **전년 동월 대비 변화율** 관점으로 해석하라.

## 시장 온도 종합 판정

| 판정 | 조건 (참고 기준) |
|---|---|
| 🟢 우호 | 금리 인하/동결 + 환율 안정 + CPI 둔화 (3개 중 3개) |
| 🟡 중립 | 2개 충족 또는 신호 혼재 |
| 🔴 비우호 | 금리 인상 사이클 또는 (환율 급등 + CPI 재가속) 동시 |

- 🔴 구간에서는 BUY-8 이 7개 이상 충족돼도 "분할 매수 1차 30%까지만"을 권고 기준으로 제시.
- 판정 근거가 된 각 지표의 **실제 수치와 기준일**을 반드시 병기하라 (`econ_latest`, `econ_series`).

## 조사 절차

```
1. econ_latest()                → 4대 지표 최신값 (기준일 확인 — 오래된 데이터면 명시)
2. econ_series(지표, 6개월)     → 사이클/추세 판정 (기준금리·국고채3년 필수)
3. 시장 온도 판정 (위 표)       → 근거 수치 병기
```

주의: 이 서비스의 데이터는 ECOS 수집 배치 주기에 의존한다. 최신값의 기준일이 현재와
크게 벌어져 있으면(예: 1개월 이상) "데이터 시점 한계"를 판정과 함께 명시하라.
