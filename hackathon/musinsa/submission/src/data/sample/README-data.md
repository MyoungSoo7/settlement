# 샘플 옷장·구매 데이터 — 심어둔 행동 패턴 정답지 (심사/검증용)

`closet.csv`(현재 옷장 16벌)와 `purchases.csv`(구매 내역 18건, 반품 포함)는 합성 데이터다.
브랜드는 전부 **가상 브랜드**(어반뮤트·스테디로우·먼데이클로젯·그레이스케일·화이트라벨·
무지톤·레이어드·테일러독·리버스데님)로, 실존 브랜드 평판에 영향을 주지 않도록 했다.
`closet-retrospective` 스킬이 아래 3개 패턴을 **스스로 발견해야** 하며,
`test/closet-data-test.mjs` 가 이 정답지의 수치 정합성을 빌드 시점에 검증한다.

## 패턴 1 — 세일 충동구매 → 미착용

| 근거 | 수치 |
|---|---|
| 할인율 55% 이상 구매 | 4건 (C04·C05·C06·C12) — 전부 trigger=세일알림 |
| 지출 합계 | 161,000원 (35,600+39,200+41,650+44,550) |
| 6개월 착용 합계 | **2회** → 착용당 80,500원 |
| 대조군 (정가 구매) | C01 베이식 후드 59,000원·38회 착용 = 1,553원/회, C13 데님 89,000원·42회 = 2,119원/회 |

기대 추론: 할인율이 구매 트리거가 될 때 착용률이 붕괴한다.
→ "세일 아이템은 장바구니 24시간 숙성 + 기존 옷과 3가지 조합이 떠오를 때만 결제" 류의 규칙 제안.

## 패턴 2 — 같은 옷 또 사기 (블랙 상의 편중)

| 근거 | 수치 |
|---|---|
| 상의 12벌 중 블랙 | **7벌 (58%)** |
| 그중 후드·스웨트셔츠 | 5벌 (C01~C05) |
| 세일 충동 4건 중 블랙 상의 | 3건 (C04·C05·C12) — 이미 5벌 있는 카테고리를 또 삼 |

기대 추론: 신규 구매가 옷장의 빈 곳이 아니라 이미 포화한 곳으로 반복 유입.
→ "구매 전 같은 색·같은 카테고리 보유 수 확인, 3벌 이상이면 다른 색/실루엣 우선" 규칙.
(크림 니트 C06 은 색은 새롭지만 착용 0회 — 색 다양화만으로는 부족하고 조합 계획이 필요하다는 반례로 사용 가능)

## 패턴 3 — 사이즈 실패 반복 (테일러독 반품 루프)

| 근거 | 수치 |
|---|---|
| 테일러독 구매 | 3회 (2025-12-02, 2026-02-10, 2026-03-15) — 전부 L |
| 반품 | 2회, 사유 모두 "사이즈 작음" |
| 3번째 구매 (C11) | 반품 대신 보유 — 6개월 착용 3회로 사실상 방치 (75,000원) |

기대 추론: 같은 브랜드에서 같은 사이즈로 3연속 실패 — 브랜드 고유의 작은 핏.
→ "테일러독은 한 치수 업(XL) 또는 상세 실측(어깨·가슴 단면) 확인 후 구매" 규칙.
반품 2회의 왕복 배송비·시간 + 방치 1벌이 비용 근거. (반품률 감소 = 플랫폼 비즈니스 임팩트와 직결)

## 대조군 (정상 행동)

- C13 리버스데님 스트레이트 데님: 정가 구매·42회 착용 — 옷장의 주력. "잘 사는 구매"의 기준선.
- C08/C09 무지톤 티셔츠 재구매: 같은 아이템 색만 2색 — 검증된 아이템의 의도적 재구매는 문제가 아님
  (패턴 2와의 구분: 착용률이 높다 — 31회/19회).

세 패턴 모두 **한 줄만 봐서는 안 보이고**, 두 파일 교차(구매 trigger×착용 횟수,
반품 이력×보유 아이템)와 시간순 재구성에서만 드러나도록 설계했다.
