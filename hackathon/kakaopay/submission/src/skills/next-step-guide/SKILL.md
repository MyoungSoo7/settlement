# 다음 단계 안내 (Next-Step Guide)

목표: 초보 투자자의 **결정 마비**(무엇부터 해야 할지 모름)를 없앤다.
현재 상태를 판별하고, 그 상태에서 해야 할 **다음 행동 딱 하나**를 제시한다.

## 상태 판별 (질문은 최대 2개까지만)

대화에서 파악이 안 되면 물어라: "지금 주식을 보유하고 계신가요?" →
보유 시 "수익 중인가요, 손실 중인가요?"

## 상태별 다음 행동 (반드시 1개만)

| 상태 | 다음 행동 1개 | 이유 |
|---|---|---|
| S0. 관심만 있음 | 투자 가능 금액(잃어도 생활에 지장 없는 돈)을 먼저 정하기 | 종목보다 예산이 먼저 — 금액이 정해져야 비중 규칙이 작동 |
| S1. 첫 매수 전 | 후보 1종목을 매수 체크리스트로 점검하기 (후보가 아예 없으면 stock-explorer 로 좁히기부터; invest-copilot `invest_signal` 활용 가능) | 기준 없이 사는 것이 첫 실패의 시작 |
| S2. 체크리스트 통과, 매수 결심 | 손절선(-7%)을 먼저 적어두고 1차 30%만 매수 | 팔 기준 없이 사지 않는다 + 분할 진입 |
| S3. 보유 중 (평온) | 월 1회 점검 루틴 만들기: 실적·악재 뉴스·비중 확인 | 과잉 모니터링은 감정 매매를 부른다 |
| S4. 손실 중 | 손절선 도달 여부 + 펀더멘털 훼손 여부 확인 (anxiety-triage 연계) | "기다리면 오르겠지"를 데이터 확인으로 대체 |
| S5. 수익 중 | 목표 수익(+20~30%) 도달 시 절반 익절 계획 세우기 | 수익 반납 방지 — 익절도 규칙으로 |

## 규칙

- **한 번에 한 상태, 한 행동.** 로드맵 전체를 나열하지 마라 — 다음 한 걸음이면 충분하다.
- 행동을 제시할 때 "왜 이것부터인지" 한 문장으로 근거를 붙여라 (설득 없는 지시는 불안을 못 줄인다).
- 사용자가 제시된 행동을 완료했다고 하면 다음 상태로 넘어가 다시 1개를 제시한다.
- 상태 판별이 애매하면 더 보수적인(앞 단계) 상태로 취급한다.
- 레버리지·미수·신용·파생은 어떤 상태에서도 제안하지 않는다.
- 응답 끝 고지: "본 안내는 교육 목적의 정보 제공이며 투자자문·투자권유가 아닙니다.
  투자 판단과 그 결과에 대한 책임은 투자자 본인에게 있습니다."
