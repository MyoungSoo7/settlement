## 1. 재고 자산 적체 - 제품군별 회전과 수주 전환 확인 필요

결론: DART 2025년 연결 재무제표 기준 재고 증가율이 매출 성장률보다 빠르게 나타났습니다. 카지노·전장 디스플레이 사업의 분할 전후 수주/출하 타이밍과 제품군별 재고 회전일수를 확인해야 합니다.

근거:
- DART 외부 신호 E2: PRESENT
- 매출 성장률: 2.7%
- 재고 증가율: 19.4%
- 매출 대비 괴리: 16.7%p
- 재고 규모: 0.1조 -> 0.1조

왜 문제인가: 재고 증가가 모두 나쁜 신호는 아닙니다. 신규 수주나 분할 이후 독립 사업 준비 과정에서 선제 생산이 발생할 수 있습니다. 그러나 매출 성장보다 재고 증가가 빠르면, 제품 믹스 변화, 고객사 납기 지연, 장기재고, 평가손실 가능성을 확인해야 합니다. 특히 카지노·전장 디스플레이는 고객사별 사양과 납품 일정의 영향이 커서 일반 재고 총액만으로는 판단이 어렵습니다.

확신도: 확인 필요. DART 수치상 경고 조건은 충족했지만, 내부 SKU별 재고와 수주잔고를 확인해야 원인을 구분할 수 있습니다.

판별 테스트: 2025년 말 재고를 카지노 디스플레이, 전장 디스플레이, 기타 제품군으로 나누고 90일 초과 재고와 확정 수주 대응 재고를 구분합니다. 확정 수주와 매칭되지 않는 장기재고가 증가하면 리스크 확신도를 높입니다.

권고 조치:
- SCM/영업: 고객사별 수주잔고와 재고 매칭표를 작성
- 재무: 재고자산 평가충당금, 폐기/할인 가능성, 분할 이후 사업별 재고 귀속 기준 점검
- CEO 보고: 제품군별 재고 회전일수와 향후 2개 분기 출하 계획을 한 장으로 정리

추가 확인 절차:
- 2025년 사업보고서 주석의 재고자산 구성 및 평가손실 확인
- 분할 전후 재고 귀속 기준과 내부거래 제거 방식 확인
- 주요 고객사별 출하 지연 또는 사양 변경 여부 확인

## 2. 공시 행간 (정정·해명 패턴) - 분할·재상장 관련 검수 체계 점검 필요

결론: 최근 90일 공시 29건 중 정정 공시가 8건 감지됐습니다. 단순 서식 정정인지, 분할·재상장·사업구조 변경 관련 핵심 내용 수정인지 분류해야 합니다.

근거:
- DART 외부 신호 E5: PRESENT
- 최근 90일 공시 건수: 29건
- 정정 공시 감지: 8건
- 해명 또는 풍문 공시: 감지 없음
- 감사 관련 플래그: 감지 없음

왜 문제인가: 인적분할과 재상장 구간에서는 공시 건수가 많아지고 정정 가능성도 커질 수 있습니다. 그러나 정정 공시가 반복되면 투자자 커뮤니케이션, 법무/IR 검수, 회계 수치 확정 프로세스 중 어디에서 수정이 발생했는지 확인해야 합니다. 특히 분할 비율, 재상장 일정, 사업부 재무정보, 주요 계약과 연결된 정정이면 단순 오기보다 내부 검수 이슈로 봐야 합니다.

확신도: 확인 필요. 플러그인은 공시 제목/메타데이터 기준으로 정정 패턴을 감지했으며, 각 정정 공시의 원문 변경 내용은 아직 비교하지 않았습니다.

판별 테스트: 정정 전후 공시 원문을 비교해 변경 항목을 단순 오기, 일정 변경, 수치 변경, 사업구조 설명 변경으로 분류합니다. 수치 또는 구조 설명 변경이 반복되면 리스크 확신도를 높입니다.

권고 조치:
- IR/법무: 최근 90일 정정 공시 8건의 변경 사유와 승인 라인을 분류
- 재무: 분할 관련 재무정보와 공시 수치의 원천 데이터를 추적
- CEO 보고: 정정 사유 중 수치·일정·구조 변경 항목만 별도 보고

추가 확인 절차:
- DART 정정 공시별 변경 전후 diff 작성
- 분할·재상장 관련 공시와 일반 정기공시를 분리해 정정률 비교
- 공시 제출 전 체크리스트와 최종 승인권자 로그 확인
