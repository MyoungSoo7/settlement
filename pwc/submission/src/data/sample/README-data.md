# 샘플 데이터 설명 (합성 데이터, 단위: 백만원)

가상 제조·유통사 "(주)한빛커머스" 의 2026Q2 기준 데이터. 세 파일은 서로 교차 검증되도록 설계됨.

| 파일 | 내용 | 기간 |
|------|------|------|
| `trial_balance.csv` | 분기별 시산표 요약 (매출·채권·재고·선수금·영업이익·영업현금흐름) | 2025Q3~2026Q2 |
| `ar_aging.csv` | 거래처별 매출채권 연령분석 — 합계는 시산표의 2026Q2 매출채권(7,200)과 일치 | 2026Q2 말 |
| `cost_allocation.csv` | 제품별 원가·공통원가 배부 — 매출 합계는 시산표의 2026Q2 매출(15,800)과 일치. 공통원가 총 3,000 을 매출 비중으로 배부 중 | 2026Q2 |

⚠️ 이 데이터에는 의도적으로 심어둔 이상 신호가 있다 (정답지는 `submission/README.md` 의 데모 섹션 참조).
데모 목적: 에이전트가 "이쁜 대시보드"가 아니라 행간의 문제를 추론으로 찾아내는지 검증.
