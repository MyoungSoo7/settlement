## 1. WBS 개요

### 1.1 목적
프로젝트의 전체 작업을 계층적으로 분류하여 일정, 자원, 비용을 체계적으로 관리한다.

### 1.2 WBS 구성 원칙
- 100% Rule: 모든 작업이 WBS에 포함되어야 함
- 최하위 작업(Work Package)은 추정·관리 가능한 단위로 분해
- 각 작업에 담당자, 기간, 산출물 명시
