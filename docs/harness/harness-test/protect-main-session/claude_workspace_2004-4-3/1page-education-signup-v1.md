## 1. 제목

## 2. 배경
분기 사내 교육을 운영하는 중에 신청 직후 환불·일정 변경 문의가 CS에 다수 유입해, 페이지 개편 방향을 합의하려고 회의를 소집했다. (← meeting.md:5)

## 3. 문제
일정·환불 안내가 페이지 맨 아래 작은 글씨로 있어 사실상 안 읽히고, 그래서 신청 후 환불·일정 문의가 CS로 들어온다. (← meeting.md:12)

## 4. 사용자
사내 교육 신청자, 그리고 문의를 받는 운영/CS팀. (← meeting.md:5, meeting.md:12)

## 5. 해결 방향
이번 범위는 두 갈래만 본다.
1. 신청 직전 핵심 정보 강조 화면 추가 (← meeting.md:22)
2. 자동 안내 메일에 환불·일정 기준 추가 (← meeting.md:23)

## 6. 성공 기준
신청 직후 환불·일정 관련 CS 문의를 다음 정기 미팅까지(약 4주) 15% 줄인다.
(감소율 15%·기간 4주는 의사결정자 ask-back 확정 2026-06-04. 목표 원문 "신청 직후 환불·일정 문의를 줄인다" ← meeting.md:10)

## 7. 확인 필요 항목 (외부 회차 대기 — 본문 단정 보류)
- 두 갈래(강조 화면·자동 메일) 각각의 최종 담당자·일정 — 월요일 정기 미팅 확정 예정 (← meeting.md:9, meeting.md:33)
