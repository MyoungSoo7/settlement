# Handoff — L1 · ask 세션 → 메인

## From task (4-1 리프)
- 목적: 회의록에서 비어 있는 결정 항목을 의사결정자 질문 표로 정리한다
- 입력: 회의록 원문 `inputs/meeting.md` (특히 `## 미결정` 단락이 단서)
- 범위 (하지 않을 일 포함): 결정권자·일정·수치 출처·환불 기준 4종 질문 + 각 질문의 대상자. 답을 짓거나 결론을 단정하지 않는다
- 금지: 회의록에 근거 없는 질문 지어내기. 답을 대신 적기. "~로 한다" 결론 톤
- 완료조건: 질문 4개 + 의사결정자(대상) 한 줄씩이 표로 정렬되고, 각 질문에 근거 줄 `meeting.md:NN`이 달린다
