## From task (4-1 리프)
- 목적: 회의록 확정 내용을 제목·배경·문제·사용자·해결 방향·성공 기준 6칸으로 맵핑하고, 미결정은 "확인 필요 항목" 한 칸으로 모은다
- 입력: 회의록 확정 묶음(`## 합의된 범위`·`## 범위 밖`·`## 발언 기록` 확정분) + L1 질문 표 + L2 보존 인용 3줄
- 범위 (하지 않을 일 포함): 7칸 = 6칸 본문 + 확인 필요 항목 1칸. L1 질문의 답은 외부 회차(월요일 미팅·운영팀 회신) 대기라 본문에 단정하지 않고 확인 필요 항목으로만 둔다
- 금지: 회의록에 없는 수치·일정·담당자·환불 기준을 결론처럼 쓰기. 단서 붙은 수치(CS 문의 60%, 출처 미확인)를 본문 6칸에 넣기. 범위 밖 항목(전체 리뉴얼·결제 모듈 모달·환불 기준 명문화)을 해결 방향에 넣기
- 완료조건: 6칸이 각각 한 줄 이상씩 채워지고, 미결정이 모두 확인 필요 항목 한 칸으로 묶인다
