# Handoff — L1 · ask 세션 → 메인

## From task (4-1 리프)
- 목적: 회의록에서 후속 결정이 필요한 항목을 질문 형태로 추출한다.
- 입력: 사내 교육 신청 페이지 개편 회의록 요약 및 원문 근거.
- 범위 (하지 않을 일 포함): 결정 질문 4개만 작성한다. 답변, 일정 확정, 담당자 지정은 하지 않는다.
- 금지: 근거 없는 사실 추가, 추정 확정, 회의록에 없는 담당자/일정 창작.
- 완료조건: 각 질문에 대상자/팀과 원문 근거가 포함되어야 한다.

## Result (이 세션이 채운 부분)
- 결과 요약:
  1. 박PM에게 질문: 최종 담당자와 일정은 다음 주 월요일 정기 미팅에서 누구로, 언제로 확정할 예정인가?
     근거: 박PM “오늘은 방향만 합의한다. 최종 담당자·일정은 다음 주 월요일 정기 미팅에서 확정한다.”
  2. 한CS에게 질문: CS 문의의 60%가 어떤 출처와 산식에서 나온 수치인지 확인 가능한가?
     근거: 한CS “정확한 수치는 다시 뽑아 와야 한다”; 60%는 출처 미확정 추정치.
  3. 운영팀에게 질문: 환불 기준 명문화 범위와 최종 확정 책임자는 누구인가?
     근거: 한CS “환불 기준 자체가 명문화돼 있지 않다”; 명문화는 운영팀 확인 필요 항목.
  4. 김디자이너/BE팀에게 질문: 디자인·메일 템플릿 최종 승인 일정이 언제 확정되며, 그 이후 BE 일정 산정은 언제 가능한가?
     근거: 김디자이너의 다음 화요일 초안은 잠정 일정이며, BE는 디자인 확정 후 일정을 산정한다고 함.
- 판정: 통과
- 남은 질문: 없음
- 다음 세션의 첫 행동: 위 4개 질문을 메인 산출물의 결정 항목 섹션에 반영한다.
