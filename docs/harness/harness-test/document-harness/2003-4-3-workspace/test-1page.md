# 사내 교육 신청 페이지 개편 — 1page 기획안 초안 v1

## 제목
사내 교육 신청 페이지 — "신청 직후 환불·일정 문의를 줄인다" v1

## 배경
- 지난 분기 신청 후 환불·일정 변경 문의가 CS로 다수 유입.
- 일정·환불 안내는 현재 페이지 맨 아래 작은 글씨로만 존재.
- 결제 모듈 모달은 외부 SDK 제약으로 이번 범위에서 손대지 않는다.

## 문제
- "일정·환불 안내가 페이지 맨 아래 작은 글씨로 있어서 사실상 안 읽힌다." (김디자이너)
- 신청 직전 환불·일정 기준을 한 번 더 강조하는 화면이 없다.
- 자동 안내 메일에도 환불·일정 기준이 빠져 있다.

## 사용자 또는 대상
- 사내 교육 신청자.
- CS팀 (현재 환불·일정 문의 1차 응대).

## 해결 방향
- (1) 신청 직전 핵심 정보 강조 화면 추가.
- (2) 자동 안내 메일에 환불·일정 기준 추가.
- "페이지 전체 리뉴얼은 이번 범위가 아니다" (박PM)

## 성공 기준
- "신청 직후 환불·일정 문의를 줄인다." (박PM, 목표 한 줄)

## 확인 필요 항목
- 최종 담당자·일정 — 다음 주 월요일 정기 미팅에서 확정 예정.
- CS 문의 60% 수치 — 한CS 추정값이며 출처 재확인 필요.
- 환불 기준 명문화 가능 여부 — 운영팀 별도 회신 대기. 현재 상태는 "환불 기준 자체가 명문화돼 있지 않다" (한CS)
- 김디자이너 시안: 다음 주 화요일까지 (잠정) — 최종 승인 미정.
- 정BE 개발 일정 산정: 시안 확정 후 3일 안 (잠정) — 최종 승인 미정.
