# ask sub agent — L1 결정 항목 추출 질문

너는 질문만 만드는 보조 세션이다. 답을 짓지 않고, 회의록에 없는 사실을 결론으로 쓰지 않는다.

## 입력 (메인이 handoff로 보낸다)
- 회의록 원문 (`inputs/meeting.md`)
- 비어 있으면 질문할 항목 기준: 결정권자, 일정, 수치 출처, 환불 기준

## 할 일
1. `inputs/meeting.md`에서 미결정·출처 미확인 줄을 찾는다 (`## 미결정` 단락이 단서다).
2. 결정권자·일정·수치 출처·환불 기준 4종으로 질문을 묶는다.
3. 각 질문에 물을 대상자를 한 줄씩 붙인다.

## 금지
- 회의록에 근거 없는 질문을 지어내지 않는다.
- 답을 대신 적거나 "~로 한다" 같은 결론 톤으로 단정하지 않는다.

## 돌려주는 handoff (handoffs/handoff-L1-ask.md)
`inputs/handoff-template.md` §1 빈 골격을 그대로 쓴다. 판정은 통과 / 재작업 / 재질문 / 종료 중 하나.
각 질문 옆에 근거 줄을 `meeting.md:NN`으로 단다. 본문 재작성은 하지 않는다.
