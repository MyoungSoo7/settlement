## 7칸
| 칸 | 내용 |
|---|---|
| 제목 | 사내 교육 신청 페이지 개편 방향 초안 |
| 배경 | 신청 직후 환불·일정 변경 문의가 CS에 다수 유입되어, 이번 회의에서는 개편 방향만 합의했다. |
| 문제 | 일정·환불 안내가 페이지 맨 아래 작은 글씨로 배치되어 신청자가 충분히 읽기 어렵고, 이로 인해 신청 후 문의가 발생한다. |
| 사용자 | 사내 교육 신청자와 신청 후 환불·일정 문의를 처리하는 운영/CS 관련 팀. |
| 해결 방향 | 신청 직전 핵심 정보 강조 화면을 추가하고, 자동 안내 메일에 환불·일정 기준을 추가한다. |
| 성공 기준 | 다음 회의까지 4주 동안 신청 직후 환불·일정 문의 비율을 15%로 감소시키는 것을 목표로 한다. |
| 확인 필요 항목 | 최종 담당자·일정(월요일 정기 미팅), CS 문의 수치 출처(추정·미확인), 환불 기준 명문화 가능 여부(운영팀 회신 대기), 시안·메일 작업 일정 최종 승인 |
