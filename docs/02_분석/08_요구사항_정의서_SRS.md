## 1. 개요

### 1.1 목적
본 문서는 Lemuel 전자상거래 주문·결제·정산 통합 시스템의 기능 및 비기능 요구사항을 상세히 정의한다.

### 1.2 범위
사용자 관리, 상품 관리, 주문 관리, 결제 관리, 정산 관리, 카테고리, 쿠폰, 리뷰 도메인의 전체 요구사항을 포함한다.

### 1.3 용어 정의

| 용어 | 정의 |
|------|------|
| CAPTURED | 결제 매입 완료 상태 |
| PENDING | 정산 대기 상태 |
| CONFIRMED | 정산 확정 상태 |
| Idempotency-Key | 멱등성 보장을 위한 고유 키 |
