## 1. 설계 개요

### 1.1 설계 원칙
- **React 18 SPA**: 단일 페이지 애플리케이션
- **반응형 디자인**: 데스크톱/태블릿/모바일 대응
- **접근성**: WCAG 2.1 AA 수준
- **컴포넌트 기반**: 재사용 가능한 UI 컴포넌트

### 1.2 프론트엔드 구조

```
frontend/src/
├── api/           # API 서비스 (axios + JWT 인터셉터)
├── components/    # 재사용 UI 컴포넌트
├── contexts/      # React Context (Toast, Cart)
├── pages/         # 페이지 컴포넌트
├── types/         # TypeScript 타입 정의
└── __tests__/     # Vitest 테스트
```

## 4. 화면 상세 설계

### 4.1 SCR-COM-001: 로그인

| 항목 | 내용 |
|------|------|
| 화면 ID | SCR-COM-001 |
| 화면명 | 로그인 |
| URL | /login |
| 접근 권한 | 비로그인 |

**화면 구성:**

```
┌─────────────────────────────────┐
│           Lemuel                │
│                                 │
│   ┌─────────────────────────┐   │
│   │  이메일                  │   │
│   └─────────────────────────┘   │
│   ┌─────────────────────────┐   │
│   │  비밀번호                │   │
│   └─────────────────────────┘   │
│                                 │

│                                 │
│   비밀번호 찾기 | 회원가입       │
└─────────────────────────────────┘
```

**이벤트 처리:**

| No | 이벤트 | 처리 | API |
|----|--------|------|-----|
| 1 | 로그인 클릭 | 이메일/비밀번호 검증 → 로그인 | POST /api/auth/login |
| 2 | 비밀번호 찾기 | 비밀번호 재설정 페이지 이동 | - |
| 3 | 회원가입 | 회원가입 페이지 이동 | - |

## 5. UI 컴포넌트 정의

### 5.1 공통 컴포넌트

| 컴포넌트명 | 설명 | Props |
|-----------|------|-------|
| Button | 기본 버튼 | variant, size, onClick, disabled |
| Input | 텍스트 입력 | type, placeholder, value, onChange |
| Modal | 모달 다이얼로그 | isOpen, onClose, title, children |
| Table | 데이터 테이블 | columns, data, pagination |
| Pagination | 페이지네이션 | currentPage, totalPages, onPageChange |
| Toast | 알림 토스트 | message, type, duration |
| SearchBar | 검색바 | placeholder, onSearch |
| Badge | 상태 배지 | status, label |

### 5.2 Context

| Context | 용도 | 주요 State |
|---------|------|-----------|
| ToastContext | 알림 메시지 관리 | toasts, addToast, removeToast |
| CartContext | 장바구니 관리 | items, addItem, removeItem, total |
| AuthContext | 인증 상태 관리 | user, token, login, logout |

### 5.3 API 서비스

| 서비스 | 용도 | 주요 메서드 |
|--------|------|-----------|
| authApi | 인증 | login, register, resetPassword |
| productApi | 상품 | getProducts, getProduct, searchProducts |
| orderApi | 주문 | createOrder, getOrders, cancelOrder |
| paymentApi | 결제 | confirmPayment, refund |
| settlementApi | 정산 | getSettlements, searchSettlements |
