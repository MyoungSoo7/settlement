## 1. 개요

| IF ID | 인터페이스명 | 유형 | 연동 대상 | 프로토콜 |
|-------|-----------|------|----------|---------|
| IF-001 | 결제 승인 | 외부 | Toss Payments | REST API |
| IF-002 | 결제 취소 | 외부 | Toss Payments | REST API |
| IF-003 | 결제 환불 | 외부 | Toss Payments | REST API |
| IF-004 | 결제 조회 | 외부 | Toss Payments | REST API |
| IF-005 | 이메일 발송 | 외부 | SMTP 서버 | SMTP |
| IF-006 | 상품 인덱싱 | 내부 | Elasticsearch | REST API |
| IF-007 | 상품 검색 | 내부 | Elasticsearch | REST API |
| IF-008 | 정산 인덱싱 | 내부 | Elasticsearch | REST API |
| IF-009 | 정산 검색 | 내부 | Elasticsearch | REST API |
| IF-010 | 프론트↔백엔드 | 내부 | React ↔ Spring Boot | REST API |

## 2. 외부 인터페이스

### 2.1 IF-001: Toss Payments 결제 승인

| 항목 | 내용 |
|------|------|
| IF ID | IF-001 |
| 연동 대상 | Toss Payments |
| 방향 | Lemuel → Toss |
| 프로토콜 | HTTPS REST API |
| 호출 방식 | 동기 |
| 인증 | Basic Auth (Secret Key) |
| 타임아웃 | 30초 |

**요청:**

| 항목 | 값 |
|------|-----|
| Method | POST |
| URL | https://api.tosspayments.com/v1/payments/confirm |
| Content-Type | application/json |

**요청 파라미터:**

| 필드 | 타입 | 필수 | 설명 |
|------|------|------|------|
| paymentKey | String | Y | 결제 키 |
| orderId | String | Y | 주문 ID |
| amount | Long | Y | 결제 금액 |

**응답 파라미터:**

| 필드 | 타입 | 설명 |
|------|------|------|
| paymentKey | String | 결제 키 |
| orderId | String | 주문 ID |
| status | String | 결제 상태 |
| method | String | 결제 수단 |
| totalAmount | Long | 결제 금액 |
| approvedAt | String | 승인 일시 |

---

### 2.2 IF-002: Toss Payments 결제 취소

| 항목 | 내용 |
|------|------|
| IF ID | IF-002 |
| 연동 대상 | Toss Payments |
| 방향 | Lemuel → Toss |
| 프로토콜 | HTTPS REST API |
| 호출 방식 | 동기 |

**요청:**

| 항목 | 값 |
|------|-----|
| Method | POST |
| URL | https://api.tosspayments.com/v1/payments/{paymentKey}/cancel |
| Content-Type | application/json |
| Header | Idempotency-Key: {uuid} |

**요청 파라미터:**

| 필드 | 타입 | 필수 | 설명 |
|------|------|------|------|
| cancelReason | String | Y | 취소 사유 |
| cancelAmount | Long | N | 부분 취소 금액 (없으면 전체 취소) |

**응답 파라미터:**

| 필드 | 타입 | 설명 |
|------|------|------|
| paymentKey | String | 결제 키 |
| cancels | Array | 취소 내역 배열 |
| cancels[].cancelAmount | Long | 취소 금액 |
| cancels[].cancelReason | String | 취소 사유 |
| cancels[].canceledAt | String | 취소 일시 |

### 2.3 IF-005: 이메일 발송

| 항목 | 내용 |
|------|------|
| IF ID | IF-005 |
| 연동 대상 | SMTP 서버 |
| 방향 | Lemuel → SMTP |
| 프로토콜 | SMTP/TLS |
| 호출 방식 | 비동기 |

**용도:**

| 유형 | 트리거 | 내용 |
|------|--------|------|

**설정:**

| 항목 | 환경 변수 |
|------|----------|
| Host | MAIL_HOST |
| Port | MAIL_PORT |
| Username | MAIL_USERNAME |
| Password | MAIL_PASSWORD |

## 3. 내부 인터페이스

### 3.1 IF-006/007: Elasticsearch 상품 인덱싱/검색

| 항목 | 내용 |
|------|------|
| IF ID | IF-006 (인덱싱), IF-007 (검색) |
| 연동 대상 | Elasticsearch 8.x |
| 프로토콜 | HTTPS REST API |
| 인증 | Basic Auth |
| 인덱스명 | products |
| 분석기 | Nori (한글) |

**인덱싱 (IF-006):**
- 트리거: 상품 등록/수정/삭제 시
- Method: PUT /products/_doc/{id}

**검색 (IF-007):**
- 트리거: 사용자 검색 요청 시
- Method: POST /products/_search
- 쿼리: multi_match (name, description)

### 3.2 IF-008/009: Elasticsearch 정산 인덱싱/검색

| 항목 | 내용 |
|------|------|
| IF ID | IF-008 (인덱싱), IF-009 (검색) |
| 인덱스명 | settlements |
| 분석기 | Nori (한글, 판매자명) |

**인덱싱 (IF-008):**
- 트리거: 정산 생성/상태 변경 시
- Method: PUT /settlements/_doc/{id}

**검색 (IF-009):**
- 트리거: 관리자/판매자 검색 요청
- 필터: 기간, 판매자, 상태, 금액 범위

### 3.3 IF-010: 프론트엔드 ↔ 백엔드

| 항목 | 내용 |
|------|------|
| IF ID | IF-010 |
| 프로토콜 | HTTPS REST API |
| 인증 | JWT Bearer Token |
| 데이터 형식 | JSON |
| Base URL | /api |

**인증 헤더:**
```
Authorization: Bearer {access_token}
```

**JWT 인터셉터:**
- 401 응답 시 Refresh Token으로 Access Token 갱신
- 갱신 실패 시 로그인 페이지로 리다이렉트

```json
{
```

```json
{

  "data": null,
```

## 5. 인터페이스 오류 처리

### 5.1 오류 코드 체계

| 코드 범위 | 영역 | 예시 |
|----------|------|------|
| ERR_1xx | 인증/인가 | ERR_101: 토큰 만료 |
| ERR_2xx | 주문/상품 | ERR_201: 재고 부족 |
| ERR_3xx | 결제 | ERR_301: 결제 실패 |
| ERR_4xx | 정산 | ERR_401: 정산 조회 실패 |
| ERR_5xx | 시스템 | ERR_500: 내부 서버 오류 |

### 5.2 외부 연동 오류 대응

| 연동 | 오류 유형 | 대응 전략 |
|------|----------|-----------|
| Toss PG | 타임아웃 | 최대 3회 재시도, 지수 백오프 |
| Toss PG | 4xx 오류 | 에러 코드 매핑, 사용자 안내 |
| Toss PG | 5xx 오류 | 재시도 후 알림 발송 |
| SMTP | 발송 실패 | 큐 기반 재시도 |
| Elasticsearch | 연결 실패 | 캐시 폴백, 알림 |
