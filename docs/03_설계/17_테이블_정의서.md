## 3. 공통 규칙

| 규칙 | 내용 |
|------|------|
| PK | BIGSERIAL 자동 증가 |
| 생성일시 | created_at TIMESTAMP NOT NULL DEFAULT NOW() |
| 수정일시 | updated_at TIMESTAMP NOT NULL DEFAULT NOW() |
| 소프트 삭제 | deleted_at TIMESTAMP NULL |
| 금액 | BIGINT (원 단위, 소수점 없음) |
